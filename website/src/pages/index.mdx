## One binary. Every REPL. Shared namespace.

rat wraps language runtimes as MCP servers. Start a Python kernel in your terminal — then connect from Claude Desktop, VS Code, Cursor, or any MCP client. Same variables. Same session. Zero glue code.

## Four tools. That's the whole API.

<div class="not-prose mt-6">
<table class="w-full text-sm">
<thead><tr class="border-b border-slate-200"><th class="px-4 py-2 text-left text-xs font-semibold tracking-wide uppercase text-slate-400">Tool</th><th class="px-4 py-2 text-left text-xs font-semibold tracking-wide uppercase text-slate-400">What it does</th></tr></thead>
<tbody>
<tr class="border-b border-slate-100"><td class="px-4 py-2"><code>run</code></td><td class="px-4 py-2 text-slate-600">Execute code or provide input</td></tr>
<tr class="border-b border-slate-100"><td class="px-4 py-2"><code>look</code></td><td class="px-4 py-2 text-slate-600">Inspect variables, complete code, see state</td></tr>
<tr class="border-b border-slate-100"><td class="px-4 py-2"><code>tail</code></td><td class="px-4 py-2 text-slate-600">Recent activity log</td></tr>
<tr><td class="px-4 py-2"><code>ctl</code></td><td class="px-4 py-2 text-slate-600">Reset, cancel, restart</td></tr>
</tbody>
</table>

Same four tools for every language. Same four tools whether you're a human in a terminal, an LLM, or a notebook app.

## Why rat

<div class="not-prose mt-6 grid gap-4 sm:grid-cols-2">
  <div class="rounded-xl border border-slate-200 bg-white p-6 shadow-sm">
    <div class="mb-2 text-lg font-semibold text-slate-900">Shared namespace</div>
    <p class="text-sm leading-relaxed text-slate-500">Your terminal, Claude, VS Code, and Cursor all see the same variables. Load data once, use it everywhere.</p>
  </div>
  <div class="rounded-xl border border-slate-200 bg-white p-6 shadow-sm">
    <div class="mb-2 text-lg font-semibold text-slate-900">Project-scoped kernels</div>
    <p class="text-sm leading-relaxed text-slate-500"><code>rat py</code> in <code>{"~/project-a"}</code> is separate from <code>{"~/project-b"}</code>. Automatic isolation, no configuration.</p>
  </div>
  <div class="rounded-xl border border-slate-200 bg-white p-6 shadow-sm">
    <div class="mb-2 text-lg font-semibold text-slate-900">Any language</div>
    <p class="text-sm leading-relaxed text-slate-500">Python, R, bash, pi agent built in. Add any language with a YAML file and a kernel script — no Go code.</p>
  </div>
  <div class="rounded-xl border border-slate-200 bg-white p-6 shadow-sm">
    <div class="mb-2 text-lg font-semibold text-slate-900">Native MCP</div>
    <p class="text-sm leading-relaxed text-slate-500">Every kernel is an MCP server. Works with Claude Desktop, Cursor, and any MCP client out of the box.</p>
  </div>
</div>
